让科学幻想变成科学现实

Microsoft HoloLens 由协同实现全息计算的特殊组件构成： 通过高级传感器以锁步方式运行的光学系统、让每秒处理大量数据的工作变得轻松的 HPU（全息处理单元）。所有这些组件以及其他硬件，让你实现自由移动并与全息影像互动。

常见问题
感谢您对Microsoft HoloLens 商业套件的关注，我们总结了一些常见问题，以便于您更好的了解相关产品细节。
1. 什么是 Microsoft HoloLens？它是如何工作的？
答：Microsoft HoloLens 是微软第一个完全独立的、运行 Windows 10 的全息计算机设备。它完全无线（没有电线），无需连接到手机或 PC。Microsoft HoloLens 可将全息影像置于物理环境当中，让你从一个全新的视角观察这个世界。
2. 什么是全息影像？
答：全息影像是一个物体的三维影像，与真实世界的物体类似，唯一的区别在于：全息影像由光组成而不是由物质组成的。与实际物体一样，也可以从不同的角度和距离观察全息物体，但是当触摸或推动全息物体时，它不会有任何物理阻力，因为它们是虚幻的。全息影像可以是二维的，像是一张纸或一个电视屏幕一样；也可以是三维的，像是真实世界中的其他实际物体一样。通过 Microsoft HoloLens 看到的全息影像将相当逼真，你可以移动它，改变它的形状，并可随着你的交互方式或所看到的物理环境改变而变化。
3. 在 Windows 10 上支持全息影像将意味着什么？
Windows 10 是微软首个支持全息计算的平台，使用 API 可在一套独立设备上支持视线、手势、声音和环境理解。在 Windows 10 中，全息影像将作为通用 Windows 平台应用，并且所有通用 Windows 应用均可作为全息影像来运行。Windows 10 中的全息影像将带给我们一种全新的沟通、创造和探索方式。
4. 如何使用 Microsoft HoloLens 看到与现实世界混合的全息影像？
答：Microsoft HoloLens 可以生成让用户看得到的多维图像，这样用户就可以感知在物理世界的全息物体了。使用 Microsoft HoloLens 看到的全息物体可以置于你选定的物理位置上，并可根据其自己的规则进行移动，或保留在特定位置上，无论你在什么位置或哪个方向。
5. Microsoft HoloLens 为哪类人群带来更多价值？
答：我们对 Microsoft HoloLens 将来实现的各种可能性感到非常兴奋。我们相信开发人员、商业组织、设计人员和创作者都会在 Microsoft HoloLens 中找到自己的独特价值。 对于开发人员来说，Microsoft HoloLens 上搭载的 Windows 10 是一个全息平台，可帮助你开发全息计算的未来。构建于 Windows 10 之上的 Microsoft HoloLens 会成为企业变革的一个革命性工具，将转变公司、设计人员和创作者处理三维数据以将产品和信息带到生活中的方式。
6. 戴眼镜和头饰时是否可佩带 Microsoft HoloLens？
答：Microsoft HoloLens 内头环可调节以适应成人头部尺寸和形状，同时也可适应大多数眼镜架，佩戴非常舒适。不过，它的设计不适用于戴帽子和头饰，为了达到最好的使用效果，最好摘下帽子和头饰。
7. 个人用户是否可以申请购买开发者版本？
答：因为开发者版本是专门面向从事 Microsoft HoloLens 应用开发的开发者用户推出的产品，所以不面向个人用户发售。
8. 开发者版本支持哪种语言？
答：做为一款全球通用的产品，目前开发者版本的支持语言为英文。
9. 开发者版本是否提供退换货？
答：开发者版本是特殊限定版本，希望通过优惠的价格方案帮助开发机构降低研发成本。由于在价格优惠的同时减少了相应的服务内容，因此开发者版本不提供任何形式的退换货。
10. 开发者版本是否提供保修服务？
答：开发者版本是针对专业的开发者用户所提供的特殊限定版本，希望通过优惠的价格方案帮助开发机构降低研发成本。由于在价格优惠的同时减少了相应的服务内容， 因此开发者版本不提供保修服务。如果您对保修服务或企业功能有需求，我们推荐您购买商业套件，这是我们面向商业机构和开发者用户的标准通用产品。
